# 17 - Angular

### Angular Basico

Iniciando os conceitos de Angular

## Angular CLI

* Instala o Angular CLI

  `npm install -g @angular/cli`

* Inicia um projeto

  `ng new NomeDoProjeto`

* Inicia o ambiente de desenvolvimento

  `ng serve`

## Comandos no Angular

* Cria um Componente

 `ng generate component NomeDoComponente`
